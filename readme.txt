Step1:Download or clone the project.
Step2:Open eclipse or STS IDE
Step3:Add Project to IDE
Step4:On the project right click -> got to maven -> give maven update
Step5:Go to project-> Right click Run As Java Application
Step6:Use postman
Step7:Add url in post man. Select GET http request
	
To Get Product Details:-	http://localhost:8082/demo/api/getproductdetails
   
To Get Config Details:-	   http://localhost:8082/demo/api/getconfigdetails
	
Step8:Go to Authorization. Select basic Authorization
Step9:Add credentials
For Product Details:- usename:user
					  password:user123
					  
For Config Details:- usename:admin
					  password:admin123
					  
Step9:Click on Send
